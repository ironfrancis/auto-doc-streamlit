### 5 Windows 安装与首启向导

- 右键 `Trea-Setup-<version>.exe` → **以管理员身份运行** 。
- 自定义安装目录（建议英文路径，避免空格/中文）。
- 首次启动时将弹出 **模型仓库向导** ：
  - 勾选 *国内镜像* 或 *国际镜像* 。
  - 选择需要缓存到本地的模型（可稍后在设置里添加）。
- 如触发 *Microsoft Defender SmartScreen* ，点击 **更多信息 → 仍要运行** 。

> **提示** ：如果在企业网络内，请在“设置 › 网络”里配置 HTTP/SOCKS5 代理，否则模型下载将超时。

### 6 macOS 安装与首启向导

- 双击 `.dmg` ，将 **Trea.app** 拖入 **Applications** 。
- `Finder › 应用程序` 中右键打开，绕过首次 *未被认证开发者* 的 Gatekeeper 阻拦。
- Catalina (10.15) 若提示无法运行：
  - 删除新版 `Trea.app` ；
  - 安装网盘里 2024-04-25 前的旧 DMG；
  - 在 **偏好设置 › 软件更新** 里关闭 *自动更新* 。
- Apple Silicon 如需运行仅编译给 x64 的旧版，可执行：

```bash
softwareupdate --install-rosetta --agree-to-license
```

### 7 大模型接入与密钥配置

> Trea 的后端使用 **OpenAI-兼容** 调用协议；只需在 `~/.trea/config.yaml` 写入对应环境变量即可启用。

#### 7.1 Doubao-1.5-pro（火山方舟）

```yaml
provider: doubao
endpoint: https://ark.cn-beijing.volces.com/api/v3
model: doubao-1.5-pro
api_key: ARK_API_KEY
```

Doubao 支持 32 k - 128 k 上下文，费用远低于 GPT-4o，适合中文长文生成

#### 7.2 DeepSeek-V3 / DeepSeek-R1

```yaml
provider: deepseek
base_url: https://api.deepseek.com/v1     # 与 OpenAI SDK 兼容
model: deepseek-chat        # V3
# 或
model: deepseek-reasoner    # R1
api_key: DEEPSEEK_API_KEY
```

DeepSeek 的 off-peak 价格在 2025-02-26 下调 75 %，是目前最具性价比的推理服务之一

#### 7.3 ChatGPT-4o

```yaml
provider: openai
base_url: https://api.openai.com/v1
model: gpt-4o
api_key: OPENAI_API_KEY
```

#### 7.4 Claude 3.5 / 3.7 Sonnet

```yaml
provider: anthropic
base_url: https://api.anthropic.com/v1
model: claude-3.5-sonnet-20240620
# 或
model: claude-3.7-sonnet-20250215
api_key: ANTHROPIC_API_KEY
```

Claude 3.5 Sonnet 在 2024-06-21 发布，主打推理与代码能力；3.7 Sonnet 继续扩容上下文窗口与速度

> **多模型热切换** ：在 Trea 设置中勾选 *Auto Fail-Over* ，即可在请求失败时自动回退到备选模型。

### 8 进阶：代理、插件、团队协作

| 场景 | 推荐做法 |
|---|---|
| 跨境网络 | 设置 ALL_PROXY=socks5://127.0.0.1:7890 或在 GUI 里填写；建议勾选 开启 DNS 代理 ，防止 api.openai.com 被污染 |
| GitHub Copilot 共存 | 在 VS Code > Extensions 禁用 Copilot 的 Inline Suggestion，避免冲突 |
| 函数调用 / 工具插件 | Doubao Pro & DeepSeek R1 均支持 Function Calling ，在“插件商店”启用后可自动解析 JSON Schema |

### 9 常见问题 & 故障排除

| 症状 | 解决方案 |
|---|---|
| macOS 10.15 打不开 | 回滚旧 DMG → 关闭自动更新；或安装 Rosetta 2 citeturn5view0 |
| Win11 启动闪退 | 删除 %APPDATA%\Trea\cache ，升级显卡驱动，再重装 |
| 请求超时 / 429 | 检查代理、API 额度；API Key 超额时可在 Billing Panel 开启 Pay-As-You-Go |
| 中文提示词乱码 | 确保文件编码为 UTF-8，无 BOM |

### 10 FAQ & 资源索引

- **如何同步配置到多台电脑？** 在「设置 › 账户」启用 *Trea Cloud Sync* ，配置文件会加密存储。
- **可以自部署私有模型吗？** 支持加载 GGUF / TensorRT 格式的本地模型，在「模型管理 › 自托管」里填写端口即可。
- **许可证与收费** 核心 IDE 保持免费；模型调用按各家 API 实际消耗计费，国内版默认绑定 Doubao 免费额度 50 万 tokens。
