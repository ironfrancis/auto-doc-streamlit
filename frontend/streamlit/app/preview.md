# Microsoft PowerToys：Windows 强力用户的效率神器（117K+ Stars）

Windows 用起来总觉得差点意思？想要更高效的窗口管理、更方便的文件操作、更智能的系统控制？

微软官方出品的 PowerToys 工具集，专为强力用户打造，现在已经开源并收获了超过 117K Stars。这套免费工具包含 30 多个实用功能，能让你的 Windows 10/11 使用体验提升一个档次。

最关键的是：**这是微软官方维护的开源项目**，安全可靠，而且完全免费。

## 这些工具到底能帮你做什么？

先说几个我认为最实用的场景：

**场景一**：你正在写代码，需要参考另一个窗口的内容，但来回切换窗口让人抓狂。用 **Always On Top**（Win+Ctrl+T），直接把参考窗口置顶，专注工作不分心。

**场景二**：从网页复制了一段文本，想转换成纯文本或者让 AI 帮你总结。用 **Advanced Paste**，粘贴的同时就能完成格式转换和 AI 处理。

**场景三**：多显示器办公，鼠标在巨大的屏幕上找不到了。**Mouse Utilities** 的 Find My Mouse 功能，按一下快捷键，聚光灯立刻锁定鼠标位置。

这只是冰山一角。接下来我会把所有 30 个工具逐一介绍清楚。

## 系统增强类工具

### PowerToys Awake - 防止休眠利器

[![A screenshot of the PowerToys Awake utility for Windows.](http://1.13.181.142:12000/i/2025/11/07/690d953a3a59e.png)](http://1.13.181.142:12000/i/2025/11/07/690d953a3a59e.png)

运行长时间任务时，最怕电脑突然休眠或者熄屏。Awake 可以让你的电脑保持清醒状态，不用去改系统的电源设置。

支持三种模式：临时保持唤醒、无限期唤醒、或者按时间计划唤醒。跑模型训练、下载大文件、演示 PPT 时特别有用。

### Light Switch - 自动切换主题

[![A screenshot of the Light Switch PowerToys utility for Windows.](http://1.13.181.142:12000/i/2025/11/07/690d9550b12ea.png)](http://1.13.181.142:12000/i/2025/11/07/690d9550b12ea.png)

根据时间自动在亮色和暗色主题之间切换。白天用亮色保护视力，晚上用暗色减少刺激，完全自动化，不用手动调整。

### Environment Variables - 环境变量管理器

[![A screenshot of the Environment Variables PowerToys utility for Windows.](http://1.13.181.142:12000/i/2025/11/07/690d9546f2cbe.png)](http://1.13.181.142:12000/i/2025/11/07/690d9546f2cbe.png)

开发者必备。Windows 原生的环境变量编辑器简直是灾难，这个工具提供了清晰的界面，还能创建配置文件来管理一组变量，在不同开发环境间快速切换。

## 窗口管理类工具

### FancyZones - 窗口布局神器

[![A screenshot of the FancyZones PowerToys utility for Windows.](http://1.13.181.142:12000/i/2025/11/07/690d954815d11.png)](http://1.13.181.142:12000/i/2025/11/07/690d954815d11.png)

这是我使用频率最高的工具。可以自定义屏幕分区布局，把窗口快速放置到预设位置。

比如：左边放代码编辑器占 60%，右边上下各放一个终端和浏览器各占 20%。设置好后，拖动窗口就能自动吸附到这些区域。多显示器用户的生产力倍增器。

### Always On Top - 窗口置顶

[![A screenshot of the Always On Top PowerToys utility for Windows.](http://1.13.181.142:12000/i/2025/11/07/690d95352a006.png)](http://1.13.181.142:12000/i/2025/11/07/690d95352a006.png)

快捷键 Win+Ctrl+T 就能把当前窗口置顶。看教程写代码、对照文档操作、监控系统状态时特别好用。

### Crop And Lock - 窗口裁剪

[![A screenshot of the Crop And Lock PowerToys utility for Windows.](http://1.13.181.142:12000/i/2025/11/07/690d9545b7b98.png)](http://1.13.181.142:12000/i/2025/11/07/690d9545b7b98.png)

可以把一个窗口的某个区域裁剪出来，创建一个新的可交互窗口。想只看某个应用的一部分内容时很实用，比如只保留视频播放器的画面部分。

### Workspaces - 工作空间管理

[![A screenshot of the Workspaces PowerToys utility for Windows.](http://1.13.181.142:12000/i/2025/11/07/690d956401bfc.png)](http://1.13.181.142:12000/i/2025/11/07/690d956401bfc.png)

可以保存当前所有应用的窗口位置和配置，一键恢复整个工作环境。

早上打开电脑，按 Win+Ctrl+`，开发环境瞬间就位：VS Code 在左边，浏览器在右边，终端在底部。不同项目可以设置不同的 Workspace，切换项目就是切换整个桌面布局。

## 快速启动与搜索工具

### PowerToys Run - 快速启动器

[![A screenshot of the PowerToys Run utility for Windows.](http://1.13.181.142:12000/i/2025/11/07/690d955aac929.png)](http://1.13.181.142:12000/i/2025/11/07/690d955aac929.png)

这是 PowerToys Run 的下一代产品，目标是成为统一的命令入口。可以访问常用命令、应用和开发工具，支持高度自定义和扩展。

### Shortcut Guide - 快捷键提示

[![A screenshot of the Shortcut Guide PowerToys utility for Windows.](http://1.13.181.142:12000/i/2025/11/07/690d955fab9ab.png)](http://1.13.181.142:12000/i/2025/11/07/690d955fab9ab.png)

按住 Win 键或者 Win+Shift+/，会显示当前可用的所有快捷键。新手学习快捷键的好帮手，老手也能发现一些被遗忘的快捷操作。

## 文件与资源管理工具

### File Explorer Add-ons - 文件预览增强

[![A screenshot of the PowerToys File Explorer add-ons for Windows.](http://1.13.181.142:12000/i/2025/11/07/690d9549769d2.png)](http://1.13.181.142:12000/i/2025/11/07/690d9549769d2.png)

在文件资源管理器中直接预览各种文件类型：Markdown、SVG、PDF、代码文件等。不用打开应用就能快速查看文件内容，大幅提升浏览效率。

整理照片、规范文件命名时效率飞升。

### Image Resizer - 图片快速调整

## 如何开始使用？

**系统要求**：
- Windows 10 2004（build 19041）或更高版本
- 或任意版本的 Windows 11
- 支持 x64 和 ARM64 架构

**安装方式**：
1. 访问 [GitHub Release 页面](https://github.com/microsoft/PowerToys/releases)下载最新版本
2. 或者通过 WinGet 安装：`winget install Microsoft.PowerToys`
3. 或者通过 Microsoft Store 安装

安装后，所有工具都在一个统一的设置界面中配置，可以根据需要启用或禁用特定功能。

## 为什么要用 PowerToys？

Windows 的默认功能确实能满足大部分日常需求，但如果你是开发者、设计师、或者重度电脑用户，PowerToys 能让你的效率提升不是一点半点。

**三个核心优势**：

1. **官方出品**：微软官方维护，安全可靠，不用担心兼容性和安全问题
2. **完全免费**：开源项目，完全免费，没有任何限制
3. **持续更新**：社区活跃，功能不断增加，bug 修复及时

而且这个项目非常欢迎贡献。如果你有好的想法或者发现了 bug，可以直接在 GitHub 上提 Issue 或 PR。

## 一些使用建议

不要一次性开启所有功能，会让系统变得复杂。建议：

1. **先从窗口管理开始**：FancyZones 和 Always On Top 是立竿见影的效率提升
2. **再加快速启动**：PowerToys Run 能替代 Windows 搜索
3. **根据工作需要选择**：开发者重点用 Environment Variables 和 Command Not Found；设计师重点用 Color Picker 和 Screen Ruler

每个工具的快捷键都可以自定义，避免冲突。
