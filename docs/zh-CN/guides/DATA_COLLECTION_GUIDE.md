## 📝 数据录入流程

### 第一步：添加频道
1. 进入"数据录入"页面
2. 选择"添加频道"标签页
3. 输入频道名称和描述
4. 点击"添加"按钮

### 第二步：添加发布记录
1. 选择"添加发布记录"标签页
2. 选择要添加记录的频道
3. 填写文章信息：
   - 标题
   - 发布日期和时间
   - 状态（已发布/草稿/已排期）
   - 数据指标（浏览量、点赞数等）
   - 文章链接
   - 标签

### 第三步：更新数据指标
1. 选择"更新数据指标"标签页
2. 选择要更新的记录
3. 输入新的数据指标
4. 点击"更新"按钮

### 手动录入
适用于少量数据或精确控制：
- 通过Web界面手动输入

### 批量导入
支持从CSV文件批量导入数据：
1. 准备CSV文件，包含以下列：
   - channel_name（频道名称）
   - title（文章标题）
   - publish_date（发布日期）
   - publish_time（发布时间）
   - status（状态）
   - views（浏览量）
   - likes（点赞数）
   - comments（评论数）
   - shares（分享数）
   - url（链接）
   - tags（标签）

2. 在"数据录入"页面选择"导入数据"标签页
3. 上传CSV文件
4. 点击"导入数据"按钮

## 🔄 数据更新流程

### 定期更新
建议按以下频率更新数据：
- **实时更新**：重要文章发布后立即录入
- **每日更新**：检查并更新数据指标
- **每周更新**：批量更新历史数据
- **每月更新**：数据清理和优化

### 数据验证
每次录入数据前请验证：
- 文章标题准确性
- 发布时间正确性
- 数据指标真实性
- 标签分类合理性

## 📈 数据指标说明

### 基础指标
- **浏览量**：文章被查看的次数
- **点赞数**：文章获得的点赞数量
- **评论数**：文章收到的评论数量
- **分享数**：文章被分享的次数

### 计算指标
系统会自动计算：
- **平均浏览量**：频道所有文章的平均浏览量
- **平均点赞率**：点赞数/浏览量的比例
- **互动率**：(点赞数+评论数+分享数)/浏览量
- **发布频率**：单位时间内的发布数量
