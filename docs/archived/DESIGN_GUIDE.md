# 首页设计指南 - Anthropic/OpenAI 风格

## 🎨 设计理念

本设计参考了 Anthropic 和 OpenAI 官网的视觉风格，追求温暖、优雅、专业的用户体验。

## 配色方案

### 主背景色
- **报纸色背景**: `#F5F1E8` - 温暖的米黄色，类似高级报纸的质感
- **侧边栏**: `#FAFAF8` - 稍浅的米色

### 卡片配色（大地色系）
1. **赤陶橙**: `#E8957B` → `#D97A5E` - 内容创作类
2. **温暖米色**: `#D4C5B0` → `#C4B19D` - 频道管理类
3. **浅棕色**: `#C8B8A8` → `#B5A393` - 系统配置类
4. **橄榄棕**: `#A3957F` → `#8F8169` - 工具类
5. **暖沙色**: `#D9B89A` → `#C9A282`
6. **奶油色**: `#E5D4C1` → `#D4C2AD`
7. **灰褐色**: `#B8A89A` → `#A89688`
8. **肉桂色**: `#CEB5A0` → `#BDA38C`

### 文字颜色
- **主标题**: `#2B2B2B` - 深灰黑色，优雅而清晰
- **副标题**: `#6B6B6B` - 中灰色
- **正文**: `#5A5A5A` - 柔和灰色

## 设计特点

### 1. 温暖的视觉体验
- 使用米黄色背景营造温暖、舒适的氛围
- 大地色系卡片与背景完美融合
- 避免使用过于鲜艳或刺眼的颜色

### 2. 极简主义
- 简洁的排版，充足的留白
- 柔和的阴影效果（不是强烈的投影）
- 微妙的边框（`rgba(0, 0, 0, 0.08)`）

### 3. 优雅的交互
- 流畅的过渡动画（cubic-bezier 缓动函数）
- 悬停时轻微上浮（4px）
- 透明按钮覆盖，点击整个卡片

### 4. 可读性优先
- 合理的字体大小和行高
- 高对比度的文字颜色
- 清晰的信息层级

## 与 Anthropic/OpenAI 的相似之处

### Anthropic 风格元素
✅ 温暖的米色/奶油色背景
✅ 大地色系（赤陶色、米色、浅棕色）
✅ 简洁的卡片设计
✅ 柔和的阴影和边框
✅ 手绘风格图标（emoji）

### OpenAI 风格元素
✅ 现代简约的布局
✅ 优雅的排版和间距
✅ 专业的视觉呈现
✅ 清晰的功能分类

## 技术实现

### Streamlit 框架下的实现
1. **全局背景**: 通过 `.stApp` 类设置整体背景色
2. **卡片样式**: HTML + CSS 实现自定义卡片
3. **交互**: 透明按钮 + `st.switch_page()` 实现点击跳转
4. **响应式**: 媒体查询适配移动端

### 核心 CSS 类
- `.main-title` - 主标题
- `.subtitle` - 副标题
- `.category-title` - 分类标题
- `.card-container` - 卡片容器
- `.card-gradient-*` - 不同的卡片配色
- `.info-panel` - 信息面板
- `.stats-box` - 统计信息框

## 使用建议

### 适用场景
✓ 内容创作和管理平台
✓ AI/ML 工具和服务
✓ 文档和知识管理系统
✓ 企业级应用首页

### 最佳实践
1. 保持充足的留白，避免拥挤
2. 使用大地色系，保持视觉一致性
3. 文字清晰可读，避免低对比度
4. 动画流畅自然，不要过于突兀
5. 响应式设计，兼顾移动端体验

## 配色参考

如需调整配色，建议在以下范围内选择：

**温暖色调**
- 米色: `#F5F1E8`, `#EAE4D8`, `#E0D5C7`
- 赤陶色: `#E8957B`, `#D97A5E`, `#C86A4E`
- 暖棕色: `#C4B19D`, `#B5A393`, `#A89688`

**冷色调（慎用）**
- 如需使用冷色调，建议降低饱和度
- 避免纯蓝、纯绿等颜色
- 可使用灰蓝色、灰绿色作为点缀
